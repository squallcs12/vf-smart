What it does

Everything your VF3 was missing

Connects to your car's control bus via ESP32 and exposes every feature through a local WiFi REST API and a companion Android app.

📱

Android Companion App

Real-time dashboard with ODO instrument cluster mode, quick actions, full controls, and Android Auto support.

🌐

WiFi REST API

Full HTTP API with WebSocket real-time updates. Control every car function from any device on your local network.

🎙️

Google Assistant

"Hey Google, lock my car." Voice commands via Android Auto App Actions — no cloud required beyond your phone.

🔒

Smart Lock Automation

Auto-close windows on lock, fold mirrors on power-off, re-open mirrors on power-on. All configurable.

⚡

Charging Monitoring

Real-time charging detection, automatic charger-port unlock when charging stops, battery voltage telemetry.

🛡️

Safety Reminders

Headlight reminder at night while in Drive, 50 ms sensor loop for immediate response, noise-filtered ADC readings.

🔄

OTA Firmware Updates

Upload new firmware or filesystem images over HTTP — no USB cable needed. Supports ArduinoOTA and PlatformIO.

🗺️

Navigation on ODO Screen

Reads Google Maps turn-by-turn from notifications and displays current maneuver and distance on the instrument cluster.

🔍

Auto Device Discovery

UDP broadcast on port 8888 lets the app find the ESP32 automatically — no manual IP entry required.

All POST endpoints require an API key in the X-API-Key header. GET endpoints are open.

Get started

Firmware quick start

Requires PlatformIO and an ESP32 development board.

1

Clone the repo

git clone https://github.com/squallcs12/vf-smart.git
cd vf3-smart
2

Build and flash firmware

# Upload filesystem (web assets)
pio run --target uploadfs

# Upload firmware
pio run --target upload
3

Complete onboarding

On first boot the device creates a WiFi AP VF3-SETUP (password: setup123). Connect and visit http://192.168.4.1 to set your WiFi credentials and API key.

4

Test the API

# Get car status
curl http://<device-ip>/car/status

# Lock the car
curl -X POST http://<device-ip>/car/lock \
  -H "X-API-Key: YOUR_API_KEY"
